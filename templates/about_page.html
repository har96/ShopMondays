Mondays is an application designed to permit users to sell stuff to
each other. After you log in, you can send messages to other users, sell
items to them, send out requests to let sellers know what your looking for,
and buy cool items from other users. Everything except for purchasing items
is FREE! Have fun!

Mondays is a webapp created by Harrison Hunter using Python and Google App Engine.
I created this site using what I learned from a Udacity course on Web Application Programming.
I've really enjoyed creating this site so I hope you enjoy using it.
